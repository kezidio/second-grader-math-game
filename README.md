# 🧮 Math Quiz Program

A simple command-line math quiz that lets users practice basic arithmetic operations: addition, subtraction, multiplication, and division.

---

## 🚀 How It Works

The program prompts the user to choose an operation:

```
1 = addition
2 = subtraction
3 = multiplication
4 = division
```

The user selects an option (1–4) or enters `-1` to exit the program.

After selecting an operation, the program generates a math question. The user must input the correct answer to proceed.

---

## ✨ Features

* Supports 4 operations:

  * Addition
  * Subtraction
  * Multiplication
  * Division
* Immediate feedback:

  * ✅ Correct answer → “Correct, good work!”
  * ❌ Incorrect answer → prompts retry
* Continuous input until the correct answer is given
* Option to exit anytime
